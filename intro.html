CSS - Cascading Style Sheets

CSS is a programming language used to improve the presentation of a html page

CSS will help modify aspects like text sizes, colors, backgrounds, image sizes and shapes, borders, positions, responsiveness(dif ferent looks on different screen sizes) among other aspects of presentation

CSS is used to design user interfaces for web pages and it helps enhance user experience.

CSS Syntax(how do we write css)

A selector is used to target an element or a group of elements for styling

{} - A declaration block - groups properties and values to be applied on the selector

Property - a presentation aspect you'd want to modify e.g. color, width,size, position

Value - the final look you are interested in for the page(red, fullpage, 400px ... )

     
        selector { 
            property: value; 
            property: value;
        } 
    
    
Example

     
        h1{ 
            color: green; 
        }
